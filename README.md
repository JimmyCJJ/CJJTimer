# CJJTimer
简单好用的定时器封装（短信、商品秒杀倒计时）(持续迭代)  
优点：高性能，自动管理内存，自动释放，高定制化，易用（支持链式语法调用）  
注：目前依赖于第三方布局库Masonry

# 使用方法
##### 手动管理
下载demo，直接把demo里的文件夹CJJTimer拖进工程即可

##### CocoaPods管理(目前最新版本为1.0.8)
```
pod 'CJJTimer'
```
